```
usage: rs [-h] [-r REPO] [--export {csv,json,xlsx,png,jpg,pdf,svg}] [--raw]
```

Display a repo summary [Source: https://api.github.com]

```
optional arguments:
  -r REPO, --repo REPO  Repository to search for repo summary. Format: org/repo, e.g., openbb-finance/openbbterminal
                        (default: None)
  -h, --help            show this help message (default: False)
  --export EXPORT       Export raw data into csv, json, xlsx (default: )
  --raw                 Flag to display raw data (default: False)
```
